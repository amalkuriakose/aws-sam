# AWS Serverless Application Model (SAM) - Example Templates

![AWS SAM Logo](AWS-SAM.jpg)

## How to run SAM templates

#### Initialize the Application:
> sam init

#### Build the Application:
> sam build

#### Deploy the Application:
> sam deploy --guided
